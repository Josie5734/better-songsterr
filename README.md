# Better Songsterr

---

A small browser add-on for Firefox which offers some improvements for the [Songsterr](songsterr.com) website

Each feature can be toggled through the popup menu when clicking on the icon in your extentions

Download from [Firefox Addons](https://addons.mozilla.org/firefox/addon/better-songsterr/)

View source code on [Github](github.com/josie5734/better-songsterr)

### Current Features:

- Automatically clicks "continue with sync pauses" on the popup that appears every so often when playing the tab to the "original" audio

- Automatically closes the banner advert at the bottom of the page
